# 분기활동 모듈

## 기능 개요
- 분기 단위 계획 수립 및 관리
- 학부모 상담 관리
- 장기 목표 설정 및 추적

## 주요 기능
1. 계획관리
   - 분기목표 설정 도우미
   - 분기목표 요청
   - 장기적인 성장전망
   - 주간목표 분석
   - 학교생활 도우미

2. 학부모상담
   - 성적관리
   - 일정관리
   - 과제관리
   - 도전관리
   - 상담관리
   - 상담앱 활용
   - 상담지연 관리
   - 다음 분기 시나리오 관리

## 개발 가이드
이 모듈은 분기별 활동에 특화된 독립적인 기능을 제공합니다.
